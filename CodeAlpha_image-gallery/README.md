# CodeAlpha Image Gallery

## 📌 Project Overview
This is a responsive Image Gallery web application developed as part of the CodeAlpha Web Development Internship.  
The project displays images in a clean grid layout with a simple and user-friendly interface.

---

## 🎯 Features
- Responsive grid-based layout for images
- Hover effects for better user interaction
- Organized image display
- Clean and minimal UI design
- Lightweight and fast performance

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (optional, if used for interactivity)
