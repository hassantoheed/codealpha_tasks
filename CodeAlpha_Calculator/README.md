# CodeAlpha Calculator

A simple and responsive calculator web application developed as part of the CodeAlpha Internship Program.

## Features

- Basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)
- Clear and reset functionality
- User-friendly interface
- Responsive design
- Fast and accurate calculations

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

```text
CodeAlpha_Calculator/
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open the `CodeAlpha_Calculator` folder.
3. Double-click `index.html` or open it in any modern web browser.
